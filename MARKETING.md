# Marketing — Stock Advisor

## Name Brainstorm

### Tier 1 — Strong candidates

| Name | Angle | Domain likely available? |
|------|-------|--------------------------|
| **Quorum** | "Algorithms reaching a quorum" — directly reflects the consensus model | quorum.io taken; quorum.app, getquorum.com likely |
| **Accord** | Agreement of signals; clean, trustworthy | accord.finance, accordapp.com likely |
| **Meridian** | The peak — signals pointing to a high point | meridian.finance possible |
| **Aphelion** | Farthest point in orbit — aspirational upside | aphelion.app likely |
| **Axiom** | Self-evident truth; precision and confidence | axiom.finance possible |
| **Confluence** | When signals converge — very literal to the model | confluence.app taken (Atlassian); getconfluence.com |

### Tier 2 — Good, broader appeal

| Name | Angle |
|------|-------|
| **Crest** | Riding the wave; peak performance |
| **Signal** | Clean, descriptive — exactly what it does |
| **Edge** | The algorithmic edge; "you have an edge" |
| **Tenor** | The direction/tone of the market |
| **Vantage** | Vantage point; see further than others |
| **Zenith** | Highest point; peak returns |

### Tier 3 — More aggressive/trading-coded

| Name | Angle |
|------|-------|
| **Apex Markets** | Clear, strong |
| **Volley** | Fast back-and-forth; options/trading feel |
| **Clearance** | Signal clarity |
| **Dispatch** | Action-oriented; send the order |

### My recommendation
**Quorum** or **Accord** — both directly reflect the consensus-of-algorithms model, are short, 
memorable, and don't scream "generic fintech." Quorum in particular has a strong conceptual hook 
("our algorithms reached a quorum: buy NVDA") that works in marketing copy and in the UI itself 
("Quorum reached — 4/5 signals bullish").

## Positioning
"The first stock advisor where algorithms have to agree before you act."

- "Five signals. One verdict."
- "Algorithmic consensus. Daily action."
